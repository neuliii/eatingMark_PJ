# EatingMark

### 나만의 맛집 찜하기
- axios 를 이용하여 서버 데이터를 불러오기.
- 찜하기 기능 추가
  
  -------------
  
### 구현 기능
▪️ 찜 하기 기능
  - 하트 버튼을 누르면 Favorite 리스트에 추가가 되며 삭제도 가능하다
▪️ 삭제 확인 모달창
  - 채워져 있는 하트를 다시 누르면 해당 찜하기 기능을 해지 할 것인지 확인하는 모달창 팝업 기능
▪️ 위치 기반 정렬
  - 브라우저를 통해 사용자의 위치를 받아서 그 기준으로 가까운 순서대로 정렬해주는 기능
